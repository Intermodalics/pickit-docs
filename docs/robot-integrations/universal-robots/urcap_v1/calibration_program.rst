Example calibration program
===========================

This :ref:`robot-camera calibration <robot-camera-calibration>` example program requires the **Pickit URCap** plugin to be installed in your robot.
For installation instructions of both the URCap plugin and the example programs, please refer to the :ref:`universal-robots-urcap-v1-installation` article.

In the initialization screen, select :guilabel:`Program robot` and then :guilabel:`Load program`. Navigate to the ``pickit_samples/robot_camera_calibration`` folder and open the ``multi_poses_calibration.urp`` program.
